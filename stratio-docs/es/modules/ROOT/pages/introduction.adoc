= Introducción

_Stratio KEOS_ tiene la capacidad de desplegarse en diversos proveedores de _Cloud_. Actualmente, de forma nativa, se soporta tanto xref:cloud-provisioner:architecture.adoc#_GCP[Google Cloud Platform (GCP)] en modo no-gestionado, como Amazon Web Services (AWS) en modo gestionado con xref:cloud-provisioner:architecture.adoc#_EKS[Elastic Kubernetes Service (EKS)].

La versión de _Stratio KEOS_ en los proveedores de _Cloud_ es la 0.1.0.
